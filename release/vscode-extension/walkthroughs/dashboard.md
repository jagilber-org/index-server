## Dashboard

1. Enable the dashboard: set `index.dashboard.enabled` to `true` in settings
2. Run **Index: Open Dashboard**
3. Browse analytics, health, and session tracking at `http://localhost:8787`

The dashboard provides real-time monitoring of your instruction index.
