# Changelog

All notable changes to the Index Server VS Code extension.

## [1.20.1] - 2026-04-16

### Changed
- Version synced with Index Server 1.20.1

## [1.13.0] - 2026-03-27

### Added
- **MCP Server Definition Provider** — Extension now registers as an MCP server provider, discoverable via `@mcp` search in VS Code Extensions sidebar
- **Auto-configuration** — Server settings are automatically applied from VS Code configuration
- **Config reactivity** — MCP server definition refreshes when settings change
- **Marketplace publishing** — CI/CD pipeline for VS Code Marketplace and Open VSX

### Changed
- Version synced with Index Server (1.13.0)

## [1.6.4] - 2025-10-15

### Added
- Initial VS Code extension release
- Commands: Configure MCP Client, Show Status, Open Dashboard
- Settings for server path, instructions directory, dashboard, and mutation control
- Bundled and standalone server support
- Welcome message on first install
